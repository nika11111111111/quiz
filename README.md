# quiz
final project 
цель проекта:создать викторину для принятия участия в викторине на разнообразную тематику
состоит из  библотек и модулей
sqlite3 
Flask
os
db_scripts(сoбственный модуль)
random
при заходе на сайт выбираете викторину, нажимаете на кнопку выбрать, выбираете правильный ответ и нажимаете на кнопку ответить . В конце викторины показывается сколько правильных ответов вы сделали 
при желании вы можете начать сначала с другой викториной 
можно заполнить базу данных своими вопросами для этого выбираете db_scripts и следуете действиям в терминале
